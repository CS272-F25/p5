About Pet Pantry Online

A class project inspired by real pet stores, built with HTML, CSS, and JavaScript.

Pet Pantry Online is a fictional e-commerce site created for a university web development course. It is inspired by sites like large pet retailers but intentionally simplified to highlight front-end concepts.

The site demonstrates semantic HTML, responsive layouts with Bootstrap, modular CSS and JavaScript, and accessibility-aware design choices.

We focus on three main pet groups: dogs, cats, and small animals. Product data is loaded from a JSON file, and cart + review data are stored in your browser using localStorage.
